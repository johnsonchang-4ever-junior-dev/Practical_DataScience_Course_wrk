# Drug-Induced Autoimmunity Prediction Project

## Project Overview

This project demonstrates the application of machine learning techniques to predict drug-induced autoimmunity (DIA) using molecular descriptors calculated with RDKit. The work showcases a complete data science pipeline from data preprocessing to model evaluation and interpretation.

- **Data Cleaning**: Removal of non-informative columns and handling of edge cases
