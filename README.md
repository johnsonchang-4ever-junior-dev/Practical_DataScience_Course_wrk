### Assignment 1: Hotel Data Cleaning Analysis

**Weight:** Data Preprocessing & EDA Fundamentals  
**Focus:** Data cleaning, validation, and exploratory analysis

#### Learning Outcomes

- Master data cleaning techniques including outlier detection and duplicate removal
- Implement comprehensive data validation and sanity checks
- Develop temporal and geographical analysis skills
- Create professional data visualizations and insights

#### Summary

- Processed 119,392 hotel booking records with 32 features
- Implemented advanced outlier detection using statistical methods
- Applied business rule validation for data integrity
- Generated actionable insights for hotel booking patterns across multiple years

### Assignment 2: Drug Autoimmunity ML Classification

**Weight:** Machine Learning Implementation  
**Focus:** Classification modeling and pharmaceutical data analysis

#### Learning Outcomes

- Apply machine learning algorithms to real-world pharmaceutical problems
- Implement feature selection and correlation analysis techniques
- Develop domain expertise in computational toxicology
- Master model comparison and evaluation methodologies

#### Summary

- Built predictive models for drug-induced autoimmunity using molecular descriptors
- Implemented K-Nearest Neighbors and Decision Tree classifiers
- Applied correlation-based feature selection on 200+ molecular features
- Achieved optimal performance through systematic parameter optimization

**Weight:** Advanced Algorithm Implementation

#### Learning Outcomes
